# 🏨 Sistema de Gerenciamento de Hotel

Projeto desenvolvido para a disciplina de **Algoritmos e Programação I (APR1)** do curso de **Bacharelado em Engenharia de Software** do **IFSP – Campus São Carlos**.

O sistema foi desenvolvido em **Python** com o objetivo de simular o gerenciamento de um hotel, permitindo o controle de clientes, apartamentos e reservas por meio de um menu interativo em terminal.

---

## 📌 Funcionalidades

- 👤 Cadastro de clientes
- 🛏️ Cadastro de apartamentos
- 📅 Cadastro e gerenciamento de reservas
- 🔍 Consulta de disponibilidade dos apartamentos
- ✏️ Atualização de registros
- 🗑️ Exclusão de informações
- 📂 Persistência de dados utilizando arquivos `.txt`
- 📊 Geração de relatórios

---

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/Jotebs/projeto_hotel.git
```

Entre na pasta do projeto:

```bash
cd projeto_hotel
```

Execute o programa:

```bash
python main.py
```
