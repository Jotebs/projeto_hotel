🏨 Sistema de Gerenciamento de Hotel

Projeto desenvolvido para a disciplina de Algoritmos e Programação I (APR1) do curso de Bacharelado em Engenharia de Software - IFSP São Carlos.
O sistema foi desenvolvido em Python com o objetivo de simular o gerenciamento de um hotel, permitindo o controle de clientes, apartamentos e reservas através de um menu interativo no terminal.

📌 Funcionalidades

👤 Cadastro de clientes
🛏️ Cadastro de apartamentos
📅 Gerenciamento de reservas
🔍 Consulta de disponibilidade
📝 Atualização de registros
🗑️ Exclusão de informações
📂 Persistência de dados utilizando arquivos .txt
📊 Geração de relatórios


🛠️ Tecnologias Utilizadas

Python 3
Programação Estruturada
Manipulação de Arquivos
CRUD
Funções
Listas
Estruturas Condicionais
Estruturas de Repetição
